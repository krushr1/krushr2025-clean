Icon Usage Guidelines

Best Practices

  • Use consistent sizing (text-lg, text-xl, text-2xl)
  • Apply semantic colors that match function
  • Maintain proper contrast ratios
  • Use hover states for interactive elements
  • Provide clear accessibility labels

Avoid

  • Mixing CS-Interface with other icon sets
  • Using decorative colors without meaning
  • Inconsistent sizing within the same context
  • Icons without proper semantic context
  • Overusing icons where text is clearer
